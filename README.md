# Site de casamento Vitor e Thamiris

> Projeto focado na criação de um site que contemplava com todos os requisitos funcionais e não funcionais que foram elaborados pelo ccasal e compreedidos
### 🚀 Status do Projeto: Concluído

---

## ⚙️ Funcionalidades (Features)

*   **Relógio:** Contabiliza o tempo até a data do casamento.
*   **História do casal:** Uma linha do tempo sobre a história do casal.
*   **Confirmação de presença:** Quando a pessoa clica no botão de confirmação de presença vai para o whatsapp do casal.
*   **Lista de presentes e pagamento via pix:** Links para lista de presença e pagamento via pix.
